### 🎮 About the Game

> Trump runs. Biden comes. You shoot rockets. You collect dollars. The wall scrolls. America wins.

**Controls:**
| Action | Key |
|---|---|
| Jump / Double Jump | `SPACE` or `↑` |
| Fire Rocket 🚀 | `F` key or tap right side |
| Collect 💵 | Just run into them |

**Enemies:** Biden 😎 · Obama 🕶️ · Hillary 👘  
**Power-ups:** Big Mac 🍔 restores a life
